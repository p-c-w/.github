## **프로젝트 소개**

### 기획 배경

- 매년 증가하는 OTT 서비스와 사용자 수 광범위하게 흩어져있는 컨텐츠들
- 보지 않는 구독서비스 유지로 인한 현명한 소비에 대한 니즈 증가
- 누가 내 OTT 구독을 관리해주면 좋을 것 같아!

<br>

### 국내 서비스중인 OTT 서비스들의 컨텐츠들을 한 곳에서 모아볼 수 있어요

- `Netfix`, `Disney+`, `Wavve`, `Watcha`, `AppleTV+`, `AmazonPrime` 의 TV, 영화 컨텐츠 들을 한곳에서 모아 볼 수 있어요
- 검색기능으로 원하는 컨텐츠의 상세정보를 찾아볼 수 있어요
- 필터로 원하는 OTT 서비스들의 컨텐츠만 찾아볼 수 있어요

<br>

### 나의 Universe로 지금 보고있는 컨텐츠, 보고 싶은 컨텐츠, 히스토리를 관리해요

- 나의 등록 정보를 기반으로 현명한 구독 관리를 할 수 있어요
- 가장 현명한 가격으로 구독할 수 있는 방법을 추천해줘요
- 현재 보고 있지 않은 구독 서비스를 알려줘요
- 나의 등록 정보를 기반으로 데이터를 기반으로 분석 그래프로 보여줘요

<br>

## **ETC**

### **커밋 컨벤션**

| Type     | 설명                                                                               |
| -------- | ---------------------------------------------------------------------------------- |
| Feat     | 새로운 기능 추가                                                                   |
| Fix      | 버그 수정 또는 typo                                                                |
| Refactor | 리팩토링                                                                           |
| Design   | CSS 등 사용자 UI 디자인 변경                                                       |
| Comment  | 필요한 주석 추가 및 변경                                                           |
| Style    | 코드 포맷팅, 세미콜론 누락, 코드 변경이 없는 경우                                  |
| Test     | 테스트(테스트 코드 추가, 수정, 삭제, 비즈니스 로직에 변경이 없는 경우)             |
| Chore    | 위에 걸리지 않는 기타 변경사항(빌드 스크립트 수정, assets image, 패키지 매니저 등) |
| Init     | 프로젝트 초기 생성                                                                 |
| Rename   | 파일 혹은 폴더명 수정하거나 옮기는 경우                                            |
| Remove   | 파일을 삭제하는 작업만 수행하는 경우                                               |
